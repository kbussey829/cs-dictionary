# Tech Dictionary

## Purpose

When I first started a Computer Science degree, I had no prior technical experience and found it difficult to learn how to "speak" the language of tech that came so easily to my peers. I'm creating this resource for anyone else that is struggling or has struggled to learn the words that seem to come so easily to everyone else. 

## Usage

The website provides terms covering a variety of areas in tech. If you notice that something is missing, submit the form on the website and I'll get to it as soon as I can. 

The definitions provided on this website are all based on my own understanding of the terminology. If you feel that one is incorrect or incomplete, submit the form on the website and check the "this definition is incorrect or incomplete" box. 

## Contributing

This resource is as much a collection of my knowledge as it is a community reference. Therefore, pull requests will not be accepted.
